# Store assets & metadata

This folder contains **text drafts** for App Store / Google Play store listings.

## Metadata
- iOS: `store/metadata/ios/`
- Android: `store/metadata/android/`

## Screenshots & graphics (to be added)
Add the following assets (not included in repo):

### iOS
- App icon: 1024×1024 PNG
- iPhone screenshots (6.7", 6.5", 5.5" etc depending on target devices)
- Optional preview video

### Android
- App icon: 512×512 PNG
- Feature graphic: 1024×500 PNG
- Phone screenshots: 1080×1920 PNG (or Play Console accepted variants)
- Optional promo video (YouTube URL)

## Privacy policy
A markdown privacy policy template is provided at the repo root: `PRIVACY_POLICY.md`.
Host it on your website and paste the URL into App Store Connect / Play Console.
